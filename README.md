# POC: React Navigation Bottom Bar

It demonstrate how to implement a type-safe parameter-based Bottom Navigation UI.

## How to run

| Description | Command |
| :--- | :--- |
| Install dependencies | `npm install` |
| Run tests | `npm test` |
| Start Metro | `npm start` |
| Run app | `npm run android` |
